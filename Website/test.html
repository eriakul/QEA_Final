Spring Mass Damper Systems

Consider this system.

The ODE of a mass, m, connected to a wall by a spring with spring constant k and a damper with damper constant b is

$$ mx'' = -kx - bx' $$

Reorganized, that is

$$ mx'' + kx + bx' = 0 $$

Now, consider a system that contains two masses coupled together to a wall with springs and dampers.

Problem 1) Write the ODE(s) for the coupled spring-mass-damper system.

Equation for m1,

$$m_1 x_1'' = -k_1 x_1 + k_2 (x_2 - x_1) - b_1 (x_1') - b_2 (x_1' - x_2')$$

Equation for m2,

$$m_2 x_2'' = -k_2 (x_2 - x_1) - b_2 (x_2' - x_1')$$

Problem 2)* We are going to assume that we can find a solution for this coupled set of second order differential equations such that both masses are oscillating with the same frequency $ω$, but with different amplitudes. Plug in trial solutions $x_1 = C_1 e^iωt$ and $x_2 = C_2 e^iωt$ to get to a system of two equations in $C_1$, $C_2$ and $ω$.

Equation for m1,

$$m_1 (C_1 e^iωt)'' = -k_1 (C_1 e^iωt) + k_2 ((C_2 e^iωt) - (C_1 e^iωt)) - b_1 ((C_1 e^iωt)') - b_2 ((C_1 e^iωt)' - (C_2 e^iωt)')$$

Equation for m2,

$$m_2 (C_2 e^iωt)'' = -k_2 ((C_2 e^iωt) - (C_1 e^iωt)) - b_2 ((C_2 e^iωt)' - x_1')$$
